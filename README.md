# NonprofitClaw

Non-profit management for [ERPClaw](https://github.com/avansaber/erpclaw). Donor management, fund accounting, grant tracking, volunteer coordination, fundraising campaigns, and compliance. 58 actions across 8 domains.

## Features

- **Donors** -- donor profiles, giving history, donor summaries and analytics
- **Donations** -- donation recording, pledge management, tax receipt generation
- **Fund Accounting** -- unrestricted/restricted funds, fund transfers, balance reports
- **Grants** -- grant applications, expense tracking, approval workflows, utilization reports
- **Volunteers** -- volunteer registration, hours tracking, assignment management
- **Campaigns** -- fundraising campaigns, progress tracking, campaign analytics
- **Programs** -- program management, outcome tracking
- **Compliance** -- 990 data preparation, audit support

## Installation

Requires [ERPClaw](https://github.com/avansaber/erpclaw) core.

```
install-module nonprofitclaw
```

## Quick Start

```
"I'm running a nonprofit -- set me up"
```

The bot will install NonprofitClaw and guide you through setting up funds, registering donors, tracking grants, and managing volunteers.
